## 🧠 What I learned

This project helped reinforce several important front-end concepts:

- Structuring components using semantic HTML (`<main>`, `<article>`, `<picture>`)
- Writing scalable CSS using design tokens and logical properties
- Improving accessibility with proper focus states and meaningful markup
- Handling responsive images efficiently with `<picture>` and `srcset`
- Organizing styles with a clean, component-based approach
